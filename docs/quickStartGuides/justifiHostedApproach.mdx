# Embed JustiFi Hosted Onboarding and Hosted Checkout
This approach allows quick integration with minimal 

## Prerequisites

Before you begin integration, ensure you have:

### 1. JustiFi Dashboard Access
- Login credentials provided by your Customer Success representative
- Access the dashboard at [https://app.justifi.ai](https://app.justifi.ai/)

### 2. API Credentials
To generate your API credentials:
1. Log into the JustiFi dashboard
2. Navigate to **Developers**
3. Toggle **Test Mode** ON for sandbox environment testing
4. Go to **API Keys** and create a test API key
5. Store your API key securely - it determines your environment (test/production)

### 3. Environment Setup
- **Test Environment**: Use test API keys for development and testing
- **Production Environment**: Use production API keys for live transactions
- The API key environment automatically routes your requests to the correct environment

## Quick Start Integration

### Step 1: Setup JustiFi Hosted Onboarding

The following integration guide explains the few steps needed to host the JustiFi Hosted Onboarding form on your platform:\
[Hosted Onboarding integration guide](https://docs.justifi.tech/api-spec#tag/Hosted-Onboarding)

### Step 2: Submit onboarding information
Render the Hosted Onboarding form and fill in all required business data (for testing use dummy data). 
Once the form was submitted successfully JustiFi creates a sub account and associates it with the business passed to Hosted Onboarding. 
This sub account is the representation of your business for payment processing.
Once your merchant is approved the sub account status changes to enabled and the merchant can start processing payments. 
Production accounts get typically enabled 1-2 business days, test accounts get enabled right away. 

### Step 4: Process Your First Payment
JustiFi offers a Hosted Checkout solution via payment link.
See the [developer docs](https://docs.justifi.tech/checkouts/hosted-checkout) for details.
