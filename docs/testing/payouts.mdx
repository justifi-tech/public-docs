## How does it work?

Currently the JustiFi testing framework only supports successful payouts. The payouts are automatically created
daily at 14:30pm US/Central Time and will succeed.

:::note CAD Payouts
Payouts for CAD sub-accounts follow a different schedule than USD and are not on JustiFi's standard daily payout schedule. See the [Canadian Payments guide](https://docs.justifi.tech/payments/canadianPayments) for details.
:::

## Webhook Events

The events related to each test payout created will be sent to the configured webhooks on the test account.
