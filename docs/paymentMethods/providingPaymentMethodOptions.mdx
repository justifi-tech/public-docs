# Providing Payment Method Options

- **Hosted and Embedded Checkout Options:** Offer a variety of payment method options to your users via embedded web
components and JustiFi Hosted Checkout
- **Seamless API Integration:** Each payment method type is supported via the same APIs as detailed in our
[API documentation](https://docs.justifi.tech/api-spec#tag/Payment-Methods).
- **Compliance and Security:** Adherence to PCI DSS and other relevant standards is crucial, especially for card-based
transactions.

By offering these diverse payment methods, JustiFi ensures that your business can cater to various customer preferences
and transaction scenarios, all while maintaining security and efficiency.
