# Payment Methods

JustiFi supports a variety of payment methods via API, Hosted Checkout and embedded web components to cater to diverse
transactional needs. The following sections provides an overview of the three primary methods:

- [Automated Clearing House (ACH) payments](achPayments.mdx)
- [Card Not Present (CNP) payments](cardNotPresent.mdx)
- [Card Present payments](cardPresent.mdx)
